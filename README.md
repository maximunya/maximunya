### 🛠️ A few things I've built

- **Pairbroker API + SDK**: a microservice that orchestrates accounts and proxies for outbound calls to dozens of external APIs (rotation, cooldowns, pooling, runtime-configurable monitoring). Sped up a core data-enrichment function **3–4×** and cut external-API errors from **~50% to 5–10%**. Adopted by other teams through the SDK.
- **mongodq**: a declarative data quality framework for MongoDB covering schema validation, field normalization, deduplication, legacy field cleanup, Airflow/Celery scheduling, reporting, and notifications. Built in production on a **200M+ document** Threat Intelligence database. Fills a real gap: no mature open-source tooling exists for MongoDB data quality at this scale.
- **Skinity**: a player data and social platform for Valorant, covering inventory sync, store data, match history, and player identity (rank, profile data, cosmetics), with social features for creating and sharing custom showcases. I design and own the backend architecture; built by a two-person team, focused on real users.
